# BioManufacturing Lakehouse Pipeline

An end-to-end data engineering project simulating a biotech manufacturing analytics platform using:

- PySpark
- Delta Lake
- SQL
- Databricks
- Medallion Architecture (Bronze/Silver/Gold)

## Project Goals

This project demonstrates:

- Industrial data ingestion
- ETL/ELT pipelines
- Data quality validation
- Star schema modeling
- Manufacturing analytics
- Lakehouse architecture

## Architecture

Raw Manufacturing Data
→ Bronze Layer
→ Silver Layer
→ Gold Analytics Layer

## Technologies

- Python
- PySpark
- Delta Lake
- SQL
- Databricks Community Edition
- Git/GitHub
